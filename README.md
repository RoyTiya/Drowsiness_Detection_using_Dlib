# Drowsiness_Detection_using_Dlib

A Drowsiness Detection Model detects in real time if a person is sleeping, drowsy or awake. 
If it detects sleeping then it will ring a Beep sound.  
It is built using Dlib toolkit. GUI built using Tkinter.
